Introducing bmtools
===================

|Pypi| |Build| |Licence|


**bmtools** provides some tools dedicated to benchmarking.


Requirements
------------

:matplotlib: >= 3.0
:numpy: >= 1.1

Installation
------------

Clone the github repo and

.. code:: console

    $ python setup.py install

or install via Pypi

.. code:: console

    $ pip install bmtools
